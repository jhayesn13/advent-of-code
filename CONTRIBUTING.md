## How to submit changes

[Open a pull request](https://github.com/jhayesn13/advent-of-code-2025/pulls) to submit changes to this project.

Your pull request needs to meet the following guidelines for acceptance:

- The [test suite](#how-to-test-the-project) must pass without errors or warnings.
- Include unit tests. This project aspires to 100% code coverage.
- If your changes add functionality, update the documentation accordingly.
